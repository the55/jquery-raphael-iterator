many many tiny drawings with raphael.js

for nonsense only. no promises. poor support

this will fill your page with small drawings. each drawing is its own little Raphael paper object

you need a container, and a function that takes a Raphael paper object.

these are the default options: {interval: 1000, width: 100, height: 100, rowSize: 5} 

var drawIt(paper) {
  // try a few circles
};

$('div#your-container').
  raphaelIterater(drawIt, options);

you're going to have to add some of your own CSS. like this. 

#container .row {
  height: 220px
}
#container .iteration {
  width: 180px;
  height: 180px;
  margin: 10px;
}
